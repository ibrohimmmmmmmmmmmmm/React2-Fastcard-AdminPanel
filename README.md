# ⚡ AdminPanelFastCard

A modern, fast, and responsive **Admin Dashboard UI** built with React and a powerful frontend stack.  
Designed for real-world e-commerce / admin systems with clean UX and scalable architecture.

## 🧠 Features

- 📊 Modern Admin Dashboard UI
- 📱 Fully Responsive (Mobile + Desktop)
- 🌙 Dark / Light mode support
- ⚡ Fast state management with Zustand
- 🧾 Form handling with Formik + Yup validation
- 🎨 Beautiful UI components with ShadCN UI
- 🎯 Clean icons with Lucide React
- 🔄 API integration using Axios
- 🎠 Swiper for sliders/carousels
- 🌐 Theme management with next-themes
- 🧩 Modular and reusable components
- 🚀 Scalable project structure
